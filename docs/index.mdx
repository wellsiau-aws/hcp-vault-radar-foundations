## My super album app

This app allows you to keep a database of existing albums in your collection

### Example set up

1. Clone the repo
1. Add your api key.

   **Example:**

   ```
   API_SECRET="w1alrXUtnJ3fI1A7BER+S/0NxR3f3R3NC3P12K3d"
   ```

1. Run the application.
